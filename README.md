## Format

- TAMMIC

```

## Author
## Motivation
## Method
## Insight
## (One-sentence) Contribution Summary
```

- 落合先生

```
## どんなもの？
## 先行研究と比べてどこがすごい？

## どうやって有効だと検証した？
## 議論はある？
## 次に読むべき論文は？
```
